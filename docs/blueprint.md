# **App Name**: Niwaas Pro

## Core Features:

- Property Onboarding: Guide users through a step-by-step process to capture property details.
- Personalized Improvement Ideas: Suggest curated and personalized home improvement ideas based on user's property profile and goals.
- ROI Prediction: Provide data-driven ROI insights for each home improvement idea, justifying the potential value uplift.
- Project Plan Tracker: Allow users to track their ongoing or planned improvements, including timeline, budget, and progress.
- Local Contractor Directory: Connect users with verified local contractors and suppliers, filtered by specialty, location, and rating.
- Expert Chat: AI-powered expert chat tool to answer user questions and provide smart suggestions on home improvement.
- User Authentication: Firebase integration to store and manage user profile, including signup, signin and authentication, as well as linking project information

## Style Guidelines:

- Primary color: Warm orange (#E76F51) to convey energy and excitement about home improvement.
- Background color: Off-white (#FFF9F4) to create a soft and inviting atmosphere.
- Accent color: Peach (#F4A261) for highlighting important information and CTAs.
- Body and headline font: 'Inter', a sans-serif font with a modern and neutral look.
- Use clear, simple icons to represent different types of home improvements and features.
- Employ a clean and well-organized layout with clear visual hierarchy to guide users through the app.
- Use subtle animations and transitions to enhance user engagement and provide feedback.
